Description
With Eltwallet, you can:

- Create an Ethereum wallet
- Restore an Ethereum wallet from a private key (with QR code scanning)
- List ETH, ELTCOIN and custom tokens balances
- List ETH, ELTCOIN and custom tokens transactions
- Send ETH, ELTCOIN and custom tokens easily with QR code scanning
- Receive ETH, ELTCOIN and custom tokens easily with QR code scanning
- Secure your wallet via a 4 digits PIN code and biometric authentication

More features to come!
